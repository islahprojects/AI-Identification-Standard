## 🎵 Audio Specifications

- **Format**: MP3 (primary) + OGG (fallback)
- **Bitrate**: 128 kbps
- **Sample Rate**: 44.1 kHz
- **Duration**: 5-8 seconds max
- **Voice**: Clear, calm, neutral accent
- **Pacing**: Slightly slower than normal speech

## 📦 File Naming Convention

ai-id-[locale]-[state].mp3
Examples:
ai-id-en-US-rendered.mp3
ai-id-fil-PH-live.mp3
ai-id-zh-CN-text.mp3


## 🎬 Recording Guidelines

1. **Tone**: Informative, not alarming
2. **Volume**: Consistent across all recordings
3. **Background**: Silent (no music or effects)
4. **Pronunciation**: Clear enunciation of "AI" (say "A-I" or "artificial intelligence")
